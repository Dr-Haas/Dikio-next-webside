**3. Recommandations Techniques**

*   **Résolution Recommandée :**
    *   **Pour le Web / Écrans Rétina :** Minimum 2048 x 2048 pixels. Idéalement 4000 x 4000 pixels pour une polyvalence maximale (zoom, découpe).
    *   **Pour l'Impression :** 300 DPI (dots per inch) à la taille d'impression souhaitée. Par exemple, pour une impression 10x10 pouces, la résolution devrait être de 3000 x 3000 pixels.
*   **Format Optimal :**
    *   **SVG (Scalable Vector Graphics) :** C'est le format le plus idéal pour le flat design et les formes géométriques. Il permet une mise à l'échelle infinie sans perte de qualité, ce qui est parfait pour ce type d'image.
    *   **PNG (Portable Network Graphics) :** Si un format raster est nécessaire, le PNG-24 est excellent. Il conserve la netteté des bords et les couleurs sans compression destructive, et supporte la transparence si jamais un arrière-plan transparent était désiré pour la lettre seule.
    *   **Eviter le JPG :** Le format JPEG introduit une compression qui peut générer des artefacts autour des bords nets et des aplats de couleur, ce qui est à proscrire pour un design plat et minimaliste.
