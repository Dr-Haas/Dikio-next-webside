# MCP Legalfab

Ce dossier contient les serveurs MCP utilises pour les projets Legalfab.

## Serveurs inclus

- `agent-rgpd` : contexte code + outils pour le projet `Agent-RGPD`
- `architecture` : documentation architecture/environnements
- `designer` : aide UX/UI
- `assets` : suggestions d'assets visuels
- `devmentor` : conseils de dev et architecture

La configuration est maintenant dediee a Legalfab.

## Installation

```bash
cd /Users/garyhaas/Desktop/Agent-RGPD/mcp
npm install
```

## Configuration Cursor

Copier `mcp/cursor-mcp-config.json` dans `~/.cursor/mcp.json`, puis redemarrer Cursor.

## Objectif

Cette base MCP est maintenue pour les projets Legalfab, avec des chemins locaux relies a `Agent-RGPD`.
